Make your products visible to everyone

How to publish products

1
Add or edit products in the admin panel

Use "Add Product" or "All Products" to manage your catalogue. Changes are saved to your browser instantly.

2
Click "Download products.json" below

This saves all your products into a single file named products.json.

3
Replace the file in your GitHub repo

Go to your GitHub repository → find products.json → click the ✏️ edit/upload icon → upload the new file → commit. Done! All visitors will now see your products.

Import products.json

Upload a previously exported products.json to restore or migrate a product catalogue. This overwrites all current products.

✅ products.json Downloaded!

Now commit this file to GitHub to make your products visible to all visitors:

  1. Open your GitHub repository in the browser
  2. Click Add file → Upload files (or find the existing products.json and edit it)
  3. Drag in the downloaded products.json file
  4. Click Commit changes
  5. Wait ~30 seconds for GitHub Pages to rebuild
  6. Refresh your store — your products are live! 🎉
